# BigTik
Проект собран с использованием ИИ, не надо использовать его на VPS. Изначально это был эксперимент по сборке проекта средстави OpenCode, сейчас проще переписать это все без ИИ или с минимальным использованием ИИ. Свою задачу BigTik плюс-минус выполняет. BigTik может стать локальным сервисом для просмотра любимых тиктоков. BigTik даёт привычный формат просмотра видео, только без интернета и сторонних платформ. Сохраняйте ролики не в папку на компьютере, а в локальный веб-сервис с тегами, открывайте его с телефона или пк и листайте коллекцию почти как в привычных приложениях.

## Возможности

- Регистрация и авторизация пользователей
- Загрузка и просмотр видео (до 1000 файлов за раз)
- Загрузка и просмотр изображений
- Просмотр видео в ленте (TikTok-стиль)
- Списки всех видео и изображений с поиском по тегам
- Админ-панель для управления пользователями
- Мультиязычность (русский, английский)

## Сборка и запуск

```bash
./mvnw spring-boot:run
```

Открыть: http://localhost:8080

## Админ-панель

По умолчанию создается учетная запись администратора (DataLoader). Логин и пароль: `admin:admin123`
Изменить пароль можно в профиле пользователя: http://localhost:8080/profile


## Создание нового аккаунта

1. Перейти на http://localhost:8080/register
2. Зарегистрировать пользователя
3. Авторизоваться на http://localhost:8080/login

Новый аккаунт может просматривать видео и изображения, но не может их загружать. Для получения возможности загрузки, пользователь admin должен сменить роль нового аккаунта с USER на CREATOR. Делается это в админ панели: http://localhost:8080/admin/users


## Роли пользователей

- **USER** - базовый доступ к просмотру ленты и поиску.
- **CREATOR** - может загружать и удалять свои видео и изображения.
- **ADMIN** - полный доступ к /admin/users, загрузке и удалению.
