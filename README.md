# BigTik

Может стать вашим локальным сервисом для просмотра любимых тиктоков. BigTik даёт привычный формат просмотра видео, только без интернета и сторонних платформ. Сохраняйте ролики не в папку на компьютере, а в локальный веб-сервис, открывайте его с телефона или пк и листайте коллекцию почти как в привычных приложениях.

## Возможности

- Регистрация и авторизация пользователей
- Загрузка видео (до 1000 файлов за раз)
- Просмотр видео в ленте (TikTok-стиль)
- Список всех видео с поиском
- Админ-панель для управления пользователями
- Мультиязычность (русский, английский)

## Сборка и запуск

```bash
./mvnw spring-boot:run
```

Открыть: http://localhost:8080

## Админ-панель

По умолчанию создается учетная запись администратора (DataLoader). Логин и пароль: `admin:admin123`
Изменить пароль можно в профиле пользователя: http://localhost:8080/profile


## Создание нового аккаунта

1. Перейти на http://localhost:8080/register
2. Зарегистрировать пользователя
3. Авторизоваться на http://localhost:8080/login

Новый аккаунт может просматривать видео, но не может их загружать. Для получения возможности загрузки, пользователь admin должен сменить роль нового аккаунта с USER на CREATOR. Делается это в админ панели: http://localhost:8080/admin/users


## Роли пользователей

- **USER** - базовый доступ к просмотру ленты и поиску видео.
- **CREATOR** - может загружать видео и удалять свои видео
- **ADMIN** - полный доступ к /admin/users, загрузке и удалению видео
